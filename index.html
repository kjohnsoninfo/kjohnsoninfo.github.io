Virtual Production Pipeline Engineer

That's No Moon - Unannounced Project

At That's No Moon, I'm currently working as a Pipeline Engineer on the Virtual Production team. I also support the animation and cinematics teams.

Notable Contributions:

  • Designed and integrated new tool request process within existing Slack channels, increasing accessibility and coordination between production, animation, and cinematics teams
  • Refactored folder organization for all shoot data, including writing a script to validate synchronization between local files and Perforce server

Primary Responsibilities:

  • Maintain database for over 15,000 takes, improving cross-team visibility of animation priorities and virtual production shot deliveries
  • Optimize Unreal Engine and motion capture workflows, writing scripts that automate repetitive tasks and reduce inefficiencies
  • Prepare stage for filming sessions by markering actors, calibrating ROM, and configuring recording software
  • Collaborate with producers, animators, and artists to resolve technical issues and implement feature requests
